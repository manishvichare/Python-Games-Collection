# 🔢 Number Guessing Game

## 📌 Description
This is a Python-based number guessing game where the computer randomly selects a number, and the player has to guess it correctly. The game provides hints like "too high" or "too low".

---

## 🎮 How to Play
1. Run the program
2. Enter your guess
3. The game will give hints:
   - 📈 Too High
   - 📉 Too Low
   - 🎉 Correct Guess

---

## 🧠 Features
- Random number generation
- Hint system (high/low feedback)
- Tracks number of attempts
- Simple and beginner-friendly game

---

## 🚀 How to Run
```bash
python main.py
