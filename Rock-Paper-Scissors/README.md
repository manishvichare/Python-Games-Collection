# ✊ Rock Paper Scissors Game

## 📌 Description
This is a simple Python console game where you play Rock Paper Scissors against the computer. The computer randomly selects its move, and the winner is decided based on classic rules.

---

## 🎮 How to Play
1. Run the program
2. Enter your choice:
   - Rock
   - Paper
   - Scissors
3. Computer will randomly choose its move
4. Result will be displayed instantly

---

## 🧠 Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

---

## 🚀 How to Run
```bash
python main.py
